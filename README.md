# 微信公众号开发平台
## 版本
```
SpringBoot 2.3.0.RELEASE

Mybatis Plus 3.1.2

Logback 内嵌日志组件

Oracle 12.1.0.1-atlassian-hosted

```

## 功能
```
1.restfu请求风格
2.aop + logback日志输出和记录
3.mybatis plus代码生成controller、service、mapper操作
4.外加lombaak、hutool
5.接口api文档生成
```
